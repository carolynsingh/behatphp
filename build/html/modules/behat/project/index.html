Running Behat

  1. After you have installed PHP and Composer, you may create a new Laravel project in terminal via the Composer create-project command:

composer create-project laravel/laravel{project_name}

  1. Download behat in the project

composer require behat/behat 3.*

  1. Download guzzle

composer require guzzlehttp/guzzle

4.To check the version of behat

vendor/bin/behat -V

  1. To initialise the project

vendor/bin/behat --init

  1. To execute the project

vendor/bin/behat

  1. Create new file in features

  1. Write the scenario in the file created

Feature: List of announcements
 Scenario: I want a list of announcements
     Given I am an unauthenticated user
     When I request a list of announcements from "http://localhost:8080"
     Then The results would include an announcement with ID "7467474"

  1. Run the project

vendor/bin/behat

  1. Create context and copy and paste these methods in feature context file

  1. Check in announcements-list.feature, feature exists and it turns to blue colour

12.Execute the project , it should show error because of pending exception

vendor/bin/behat

  1. Import pending exception to remove fatal error

use Behat\Behat\Tester\Exception\PendingException;

  1. Execute the project again, highlighted text will change its colour from red to yellow

  1. Make a call to API

$client = new GuzzleHttp\Client(['base_uri'=>'http://localhost:8080']);

     $response = $client->get (uri:'/');

     $responseCode = $response->getStatusCode();

     if ($responseCode !=200) { throw new Exception (message:"Not able to access!");
     }

     return true;

  1. Execute the project again,”Given I am an unauthenticated user” will change its colour from yellow to green showing it is verified
