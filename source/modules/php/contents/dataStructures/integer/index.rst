Integer
=========

An integer data type is a non-decimal number between -2,147,483,648 and 2,147,483,647.

In the following example $x is an integer.

The PHP var_dump() function returns the data type and value:

.. code-block:: bash

   <?php
   $x = 5985;
   var_dump($x);
   ?>

Result : int(5985)
